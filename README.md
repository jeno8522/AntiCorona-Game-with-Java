# AntiCorona-Game-with-Java

방향키(↑,↓,→,←)로 작동하는 간단한 게임

움직이는 virus와 접촉하면 흡수하면서 커짐

stage는 한국, 중국, 일본 3개로 이루어져있고 클리어하면 다음 스테이지로 넘어감

각종 아이템, 갑자기 커지거나 불규칙하게 움직이는 virus 들이 존재함
